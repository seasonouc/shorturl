短链接服务文档说明

接口： /v1/generateConf  请求方式：post ，说明：自定义短链接生成方式

| 参数     | 是否为空 | 说明                                |
| -------- | -------- | ----------------------------------- |
| longUrl  | 否       | 要转换的长链接                      |
| shortUrl | 可为空   | 自定义的短链接                      |
| length   | 可为空   | 大于12 ，小于32，自定义短链接的长度 |

接口：/v1/generate 请求方式 :get ，说明：用默认方式生成短网址

| 参数    | 是否为空 | 说明           |
| ------- | -------- | -------------- |
| longUrl | 否       | 要处理的长网址 |

接口：/v1/getVisitTimes 请求方式 :get 说明：获取某个短网址被访问的次数

| 参数     | 是否为空 | 说明             |
| -------- | -------- | ---------------- |
| shortUrl | 否       | 需要查询的短网址 |
